# 서비스 소개

>**PicMate**는 PicMate는 복잡한 사진 정리를 쉽고 빠르게 도와주는 스마트 사진 정리 앱입니다. <br>
>사진이 너무 많아 정리할 엄두가 나지 않는 분들을 위해, PicMate는 월별 자동 분류, 즐겨찾기 관리, 불필요한 사진 삭제, 앨범 정리 기능을 제공합니다.<br>
>심플하고 직관적인 UI로 누구나 쉽게 사용할 수 있으며, 원하는 사진만 쏙쏙 골라 정리할 수 있어 시간과 공간을 절약할 수 있어요. <br>
>정리되지 않은 사진들로 앨범이 복잡해졌다면, 지금 바로 PicMate로 깔끔하게 시작해보세요! <br>
>📷 쉽고 빠른 사진 정리, PicMate 하나면 충분해요! 📷
